## Чем занимался и где посмотреть результат

- Участвовал в разработке сайтов, которые продаются за миллионы рублей и попадают на различные выставки, например: https://zao-sms.ru/ и https://z1light.ru/. За время своих обязанностей успел создать фичи и пофиксить баги большинства клиентских сайтов, а также поучаствовал в создании пары проектов с нуля. Подробнее можно увидеть [здесь](https://goodfellazz.ru/).
- Основная специализация — backend: работа с серверами для ПО и сложных сайтов. Для этого использовал PHP (и нативный, и Laravel), SQL (CRUD и оптимизация), Nginx (или Apache) и Linux (клиенты хостились на разных дистрибутивах). Для домашних локалок использую эмулятор Ubuntu — WSL (удобная штука, чтобы не было проблем со сканированием видеоигр из-за Linux), а также многие вспомогательные компоненты вроде Mailpit (для email-рассылок), Meilisearch (для алгоритмов поиска) и т.д.
- Также занимался вёрсткой в паре с фронтендером. Кроме стандартных HTML, CSS, JS, использовал Vue.js, jQuery, Nuxt и Tailwind. Есть опыт вёрстки по шаблонам. Например, [вот](https://goodfellazz.ru/works/glavbankrot) и [вот](https://goodfellazz.ru/works/bigpapagifts) пара страниц, которые я лично сверстал и добавил на визитку компании по Figma-шаблонам от нашего дизайнера. Конкретно это просто фронт, которому не нужно обращаться к серверу, но основная работа с клиентскими сайтами обычно происходила в связке фронта и бэка.
- На подработке помогал в развитии и поддержании [CTMS (Clinical Trial Management System)](https://inchronicles.com/) — ПО для проведения клинических исследований.
